# 🃏 Blackjack Elite Pro Assistant

Ein professioneller Strategie-Rechner und Karten-Zähl-Assistent für Live-Dealer-Blackjack (z.B. PokerStars, Evolution).

Dieses Tool kombiniert die **Basic Strategy** mit dem **Hi-Lo Card Counting System** und den **Illustrious 18 Deviations**, um dir einen mathematischen Vorteil am Tisch zu verschaffen.

## ✨ Top Features

- **🧠 Hi-Lo Counting Engine**: Echtzeit-Berechnung des Running Counts und **True Counts** (berücksichtigt die verbleibenden Decks).
- **★ Illustrious 18 Deviations**: Das Tool schlägt automatisch Abweichungen von der Basis-Strategie vor, wenn der Count hoch genug ist (z.B. Stand auf 16 gegen 10).
- **🔄 Split-Support (Multi-Hand)**: Verwalte bis zu zwei Hände gleichzeitig mit individuellen Summen-Anzeigen und Strategie-Tipps.
- **📱 Multi-Player Table Tracking**: Erfasse im "TABLE"-Modus die Karten anderer Spieler für einen präziseren Count, ohne deine eigene Strategie zu verfälschen.
- **🛡️ Auto-Save Persistence**: Alle Daten (Count, Statistiken, Karten) bleiben bei einem Browser-Refresh (f5) erhalten.
- **🎨 Midnight Casino Design**: Hochmodernes Design mit Glassmorphism-Effekten für optimale Lesbarkeit.

## 🚀 Schnellstart

1.  **Installation**: Keine nötig! Öffne einfach die `index.html` in einem beliebigen Browser.
2.  **Settings**: Wähle die Anzahl der Decks in der Sidebar aus (standardmäßig 8 Decks).
3.  **Karten eingeben**:
    - Nutze die Tastatur auf dem Bildschirm für Kartenwerte (2-A).
    - Klicke auf die Slots für Dealer/Player oder nutze den **TABLE**-Modus für Karten anderer Spieler.
4.  **Optimal Move**: Folge der großen Anzeige oben ("HIT", "STAND", "DOUBLE", "SPLIT"). Ein **★ DEVIATION ★** zeigt an, dass der Count die Standard-Strategie optimiert hat.

## 📊 Bedienung

- **NEW ROUND**: Leert den Tisch für die nächste Runde.
- **RESET SHOE**: Setzt den Count zurück (wenn der Dealer mischt).
- **RESET STATS**: Löscht deine Win/Loss Statistik.
- **UNDO**: Macht den letzten Karteneintrag rückgängig.
